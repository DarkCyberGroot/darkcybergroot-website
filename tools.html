Nmap

A powerful network scanning tool.

Nmap ("Network Mapper") is a free and open-source utility for network discovery and security auditing. Many systems and network administrators also find it useful for tasks such as network inventory, managing service upgrade schedules, and monitoring host or service uptime.

GitHub: Nmap

Wireshark

A network protocol analyzer.

Wireshark is the world's foremost and widely-used network protocol analyzer. It lets you see what's happening on your network at a microscopic level and is the de facto standard across many commercial and non-profit enterprises, government agencies, and educational institutions.

GitHub: Wireshark

Metasploit

A penetration testing framework.

The Metasploit Framework is a platform for developing, testing, and executing exploit code. It is used by cybersecurity professionals to test the security of networks and systems by simulating real-world attacks. Metasploit includes hundreds of known exploits and tools to assist with penetration testing and vulnerability assessment.

GitHub: Metasploit

SQLmap

An automated SQL injection tool.

SQLmap is an open-source penetration testing tool that automates the process of detecting and exploiting SQL injection flaws and taking over database servers. It includes a powerful detection engine, many niche features for the ultimate penetration tester, and broad range of switches lasting from database fingerprinting to data fetching from the database, accessing the underlying file system, and executing commands on the operating system via out-of-band connections.

GitHub: SQLmap

Aircrack-ng

A wireless network security tool.

Aircrack-ng is a complete suite of tools to assess WiFi network security. It focuses on different areas of WiFi security, including monitoring, attacking, testing, and cracking. The suite includes airodump-ng for packet capture, aireplay-ng for packet injection, aircrack-ng for WEP and WPA/WPA2-PSK key cracking, and airdecap-ng for decrypting WEP/WPA/WPA2 capture files.

GitHub: Aircrack-ng

Nessus

A vulnerability scanner.

Nessus is a proprietary vulnerability scanner developed by Tenable, Inc. It is used to identify vulnerabilities in computer systems and networks. Nessus performs over 1200 security checks, including a wide range of potential threats, including missing patches, default passwords, and misconfigurations.

GitHub: Nessus

OWASP ZAP

A web application security scanner.

The OWASP Zed Attack Proxy (ZAP) is one of the world's most popular free security tools and is actively maintained by a dedicated international team of volunteers. ZAP helps you find security vulnerabilities in your web applications during the development and testing phase.

GitHub: OWASP ZAP

Hashcat

A password cracking tool.

Hashcat is the world's fastest and most advanced password recovery tool. Hashcat is free and open-source software and is used to break numerous password hashes using various techniques like dictionary attacks, brute force, and rule-based attacks.

GitHub: Hashcat

PowerSploit

A PowerShell post-exploitation framework.

PowerSploit is a collection of PowerShell scripts that can be used for a variety of post-exploitation tasks during authorized penetration tests. It includes scripts for code execution, persistence, reconnaissance, and exfiltration.

GitHub: PowerSploit

LinEnum

A Linux privilege escalation script.

LinEnum is a script that enumerates common security vulnerabilities on Linux systems. It performs automated checks in areas like kernel exploits, user accounts, passwords, and permissions, making it a useful tool for penetration testers looking to escalate privileges on Linux systems.

GitHub: LinEnum
